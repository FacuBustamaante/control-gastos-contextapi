
# Control de Gastos con Context API

Aplicación de gestión de gastos desarrollada en React utilizando Context API para el manejo global del estado.

## Características

- Agregar, editar y eliminar gastos
- Visualización de listado de gastos
- Filtro de gastos por categoría
- Control de presupuesto
- Persistencia de datos en localStorage

## Instalación

```bash
git clone https://github.com/tuusuario/control-gastos-contextapi.git
cd control-gastos-contextapi
npm install
npm start
```

## Uso

1. Define tu presupuesto inicial.
2. Agrega gastos con nombre, cantidad y categoría.
3. Visualiza y gestiona tus gastos desde la interfaz.

## Tecnologías

- React
- Context API
- CSS
